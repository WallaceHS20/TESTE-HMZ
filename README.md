## Instalação

### Clone o repositório
~~~git
https://github.com/WallaceHS20/teste-desenvolvedor-frontend.git
~~~~

### Acesse a pasta por seu terminal de preferência e instale as bibliotecas
~~~CMD
cd .\dotlib\
~~~~

~~~node
npm install
~~~~

### Inicie o App
~~~node
npm run start
~~~~

### Acesse pelo seu navegador
~~~node
http://localhost:5173/
~~~~
